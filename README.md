# 小程序应用框架 wepy 的测试和开发模板,内涵测试 demo

## 使用方法

```bash shell
#安装 wepy-cli 工具
npm install wepy-cli -g

#安装依赖
npm  install

#开启实时编译并保持命令行运行
npm run dev
```

打开`微信开发者工具`-->`小程序`-->`新增`

> 注: `项目目录先择本文件根目录即可`

`project.config.json` 文件可修改 AppID 为自己的 ID,项目名称和描述

详细使用方法参见:
[wepy 官方文档](https://tencent.github.io/wepy/document.html)
`创建与使用`部分
